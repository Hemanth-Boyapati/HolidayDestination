How to run our project -Holiday Destination
1. Install the XAMPP application
Download the zip file and extract it
2. Copy the HolidayDestination folder 
3. Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)
4. Open PHPMyAdmin (localhost/phpmyadmin)
5. Create a database with name thd
6. Import thd.sql file(given inside the database folder)
7. Run the script http://localhost/HolidayDestination   (user view)
8. For admin panel http://localhost/HolidayDestination/admin  (admin panel)
